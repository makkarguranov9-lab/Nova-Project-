# Nova-Project-
Nova Project 1.0 — Mobile Brawl Stars Server (Old School)
Nova Project 1.0 — это уникальный приватный сервер для классического Brawl Stars, который можно запустить прямо на твоем Android-устройстве. Мы вернули старый интерфейс, ящики и ту самую атмосферу 2019 года.
🚀 Особенности (Features)
Unlimited Resources: На старте доступно 985,671 гемов и 127,505 золота.
Classic Boxes: Полностью рабочие Big Box и Mega Box.
Brawler Collection: Доступно 35 из 34 бойцов (включая классических Шелли, Ниту, Кольта и других).
Social System: Рабочая система клубов (Клуб: archive, Тег: #2CLJJJ2PLP) и живой чат.
Maps & Modes: Классические карты для "Ограбления" (Горячая кукуруза, Пит-стоп) и "Награды за поимку".

.
 Инструкция по запуску (Android)
Так как версии для ПК нет, сервер запускается через мобильный интерпретатор Python.
Скачайте приложение: Установите Pydroid 3 из Google Play.
Подготовьте файлы: Скачайте этот репозиторий и файл main.py на свой телефон.
Запуск:
Откройте Pydroid 3.
Выберите файл main.py.
Нажмите на желтую кнопку Run (Play).
Результат: В консоли должна появиться надпись: [INFO] Server started! Ip: 0.0.0.0, Port: 9339.
🛠 Технические детали
Язык: Python.
Порт: 9339.
Авто-конфигурация: При первом запуске скрипт автоматически создаст config.json, если его нет.
